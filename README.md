# 🏅 Notas de Atletas

Aplicação desenvolvida em JavaScript para calcular a média de notas de atletas em uma competição de ginástica artística, seguindo critérios específicos de avaliação.

---

## 📋 Sobre o projeto

Em uma competição de ginástica, cada atleta recebe 5 notas de jurados diferentes.

Para garantir justiça na avaliação:
- A **maior nota é descartada**
- A **menor nota é descartada**
- A média é calculada com base nas **3 notas restantes**

---

## ⚙️ Tecnologias utilizadas

- JavaScript (ES6)
- Node.js (para execução)

---

## 📊 Regras de cálculo

- Cada atleta possui 5 notas
- Ordenação das notas
- Remoção da maior e menor nota
- Cálculo da média das 3 restantes

---

## 🧠 Lógica aplicada

- Uso de `.sort()` para ordenação
- Uso de `.slice()` para remoção de extremos
- Uso de `.forEach()` para soma
- Estrutura de repetição `for` para percorrer atletas
